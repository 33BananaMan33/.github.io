🚀 GitHub Pages 静态部署说明
✅ 纯静态功能（无需后端）：
  • CSV 解析与数据清洗
  • Z-Score / IQR 异常检测
  • 帕累托前沿计算
  • 线性回归 What-if 仿真
  • ECharts 交互式图表
  • HTML 报告导出
⚠️ 需要额外方案的功能：
  • AI 分析：浏览器直调 DashScope API（可能受 CORS 限制）
  • 替代方案：部署 Serverless 代理函数（Vercel/Cloudflare Workers）转发 API 请求
📋 优化建议：
  1. 配置 GitHub Actions 自动部署
  2. 添加 Service Worker 实现离线缓存
  3. 使用 IndexedDB 存储历史分析记录
  4. 如需 .docx 格式 → 引入 docx.js CDN
